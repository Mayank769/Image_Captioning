Description

Image Captioning is the process of generating textual description of an image. It uses both Natural Language Processing and Computer Vision to generate the captions. With visual attention, it gives better results as compared. This Model is trained on Flickr_8k Dataset with 8091 images and 40,455 captions,5 for each images , with epoch value as 12.Result is obtained on the greedy approach(second result),and random(1st result).
